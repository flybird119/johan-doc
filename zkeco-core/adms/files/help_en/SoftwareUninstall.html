Software uninstallation


1. Software uninstallation:
If you do not need to use this software any more, and want to delete it from your computer, perform the following: Quit all our programs, enter Start –> Programs , and click [Uninstall].
This does not delete all files. You need to delete relevant folders in the software installation directory.

2. Database uninstallation:
If you do not need to use this database software any more, and want to delete it from your computer, perform the following: Quit all our programs, enter Start –> Settings –> Control Panel –> Add/ Delete Programs, select [MySQL Server 5.1], and click [Delete] button.
This does not delete all files. You need to delete relevant folders in the software installation directory.

3. Uninstallation of fingerprint reader driver:
Quit all fingerprint management system programs, enter Start –> Settings –> Control Panel –> Add/ Delete Programs, select [Fingerprint Reader Driver], and click [Delete] button.
This does not delete all files. You need to delete relevant folders in the software installation directory.
