Help Center -> Installation And Uninstallation

Before installation


1. If you install the software in Windows 7 system, please use administrator account to log in.

2. If you have installed this software before, please uninstall it first, and restart your computer.

3. It’s better to turn off all the other application program before installation, to avoid any conflict in installing procedure.

4. During the installation, the computer firewall software may prevent some service of the system, please select “Allow all”, or temporarily stop the firewall software before installation.

5. Check the system environment variables:
Right click [My computer] on desktop, click Properties - Advanced - Environment Variables, double click the variable [path], , show the following interface:

Check if the variable value [% SystemRoot% \ system32;] is exist, if not, you need to add it manually.(This variable is exist by default, but some server administrator maybe delete it for security consideration).

6. You need install the database before program installing. This document introduce the procedure of MySQL Server installation and configuration as an example.
If you have installed the other database software before, you can skip the database installation, and install the program directly. During installation, you need to input the database information as needed.
If the database is not work properly, or any fault happened, you can re-configure the database (database restoration). Such as MySQL Server 5.1, click [MySQL Server Instance Config Wizard] in the directory of Start –> Program –> MySQL –> MySQL Server 5.1. For the configuration details, please refer to the description in Database Installation. Database restoration won’t affect the original data.

7. If you don’t want to install the database in your computer, you can access database server in other computer. But you need to configure the database server forehead, especially the following items: In Security Settings tick [Enable root access from remote machines], in port number settings tick [Add firewall exception for this port ]. Please refer to the description in Database Installation.

8. During the database configuration, you need select to use [utf8] language encoding. See the specific operation please refer to Database Installation..

9. The photograph in this manual may be different from that of the real installation process. The real installation process prevails.
Put the installation CD into the CD driver, the system will automatically pop up the program selection menu. If your computer have not installed any database before, please select to install MySQL 5.1 database program (system default), after installing the database, select to install program.
Note: MySQL Server 5.1 software in CD-ROM just for demonstration, if you need to use it, please download it from official website.
