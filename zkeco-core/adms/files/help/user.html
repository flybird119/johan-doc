系统帮助 -> 系统设置 -> 用户管理

用户管理


为系统添加新的用户，分配用户角色（权限）。


新增用户：


1.点击【新增】，输入新的用户信息，其中【*】号为必填项。各参数说明如下：



用户名：不多于30个字符。只能用字母数字或者@.+-_字符；

密码：长度必须大于4位，小于18位，默认密码为111111；

设置授权部门：不选部门将默认拥有所有部门权限；

设置授权区域：不选区域将默认拥有所有区域权限；

输入名字、姓氏、E-mail 地址；

设置是否有效：指明用户是否可以登录到这个管理站点，点选则为有效；

设置是否为超级用户：指明该用户缺省拥有所有权限，点选则为超级用户，不需选择角色；

选择角色：非超级用户需要选择角色，选择某一预先设置好的角色配置，则此用户拥有该角色中配置的各操作权限。


2.编辑完成后，点击【确认】完成用户新增，并在列表显示该用户。

对已有用户可以进行修改，点击用户名后的【编辑】，进入编辑界面，修改完成后点击【确认】保存并返回。
